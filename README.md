# Sistema de Biblioteca

## Proposta

Sistema gráfico para cadastro de livros de uma biblioteca.

## Requisitos Funcionais

- Cadastrar livros
- Exibir informações dos livros
- Identificar tipo de material

## Conceitos Utilizados

- Herança
- Encapsulamento
- Polimorfismo
- Tratamento de erros
- Interface gráfica Swing

## Tecnologias

- Java
- Swing
